Greetings dear user. Please read the following usage text.

  • To get the download links for a YouTube video or playlist, paste the URL below and press Enter or click the Creep button.
  • The URL format: {optional}/watch?v={id} for video and {optional}/playlist?list={id} for playlist. So, if you are copy/pasting from address bar, that will work just fine. Channels are not supported at this moment.
  • In case of playlists, all formats for all available qualities are provided in their respective text boxes. Just copy them to a text file and import the file from a downloader like IDM, uGet etc. Details for individual videos are shown just below the text boxes. If you don't want those for some reason, check the Text only checkbox.
  • To reduce annoyance, small enhancements have been done, e.g automatic appropriate text selection, ctrl+A etc.
  • Please use, support and spread awareness about free software. Read more about free software and GNU.
